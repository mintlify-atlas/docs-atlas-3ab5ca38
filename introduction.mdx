# Introduction to AppBiblioteca

AppBiblioteca is a cross-platform library management application built with React Native and Expo. It provides a modern, intuitive interface for managing books, authors, and users across iOS, Android, and web platforms.

## What is AppBiblioteca?

AppBiblioteca is a comprehensive library management solution that allows librarians and administrators to:

- Manage book collections with detailed cataloging
- Track authors and their works
- Handle user management and authentication
- Access the library system from any device (mobile or web)
- Perform CRUD operations on library resources

The app is designed with a mobile-first approach while maintaining full web compatibility, making it accessible from anywhere.

## Key Features

### Cross-Platform Support
Built with Expo, AppBiblioteca runs seamlessly on:
- **iOS** devices (iPhone and iPad)
- **Android** smartphones and tablets
- **Web browsers** for desktop access

### Secure Authentication
- JWT-based authentication with token refresh
- Secure token storage using Expo SecureStore
- Automatic session management
- Protected routes and API calls

### Complete Library Management
- **Books**: Add, view, and manage book collections
- **Authors**: Track author information and their works
- **Users**: User management with role-based access
- **Profile**: User profile management and preferences

### Modern User Experience
- Clean, intuitive interface
- Tab-based navigation for easy access
- Real-time data synchronization
- Dark mode support (automatic based on system preferences)
- Haptic feedback for better interaction

## Who Should Use AppBiblioteca?

AppBiblioteca is ideal for:

- **Small to medium-sized libraries** looking for a modern digital solution
- **Educational institutions** managing their book collections
- **Community centers** with lending libraries
- **Book clubs and reading groups** tracking shared resources
- **Developers** learning React Native and Expo development

## Architecture Highlights

### File-Based Routing
The app uses Expo Router's file-based routing system, organizing screens into logical groups:
- `(auth)` - Authentication screens (login, register)
- `(main)/(tabs)` - Main app with tab navigation (books, authors, users, profile)

### State Management
Zustand provides a simple, scalable state management solution with minimal boilerplate. The auth store handles user authentication state globally.

### API Integration
Axios is configured with interceptors for:
- Automatic JWT token injection
- Token refresh on 401 responses
- Centralized error handling
- Base URL configuration
