## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (version 18 or higher)
- **npm** or **yarn** package manager
- **Expo CLI** (optional but recommended)

<Info>
  If you don't have Expo CLI installed globally, you can use `npx expo` commands instead.
</Info>

## Installation

<Steps>
  <Step title="Clone the repository">
    Clone the AppBiblioteca repository to your local machine.
    
    ```bash
    git clone https://github.com/ederjuarez/biblioteca_frontend-.git
    cd biblioteca_frontend-
    ```
  </Step>
  
  <Step title="Install dependencies">
    Install all required packages using npm or yarn.
    
    <CodeGroup>
    ```bash npm
    npm install
    ```
    
    ```bash yarn
    yarn install
    ```
    </CodeGroup>
    
    This will install all dependencies including:
    - React Native and Expo
    - Expo Router for navigation
    - Axios for API calls
    - Zustand for state management
    - expo-secure-store for token storage
  </Step>
  
  <Step title="Configure the API endpoint">
    Update the API base URL in `api/axios.ts` to point to your backend server.
    
    ```typescript api/axios.ts
    const api = axios.create({
      baseURL: "http://localhost:8000/api", // Change this to your API URL
    });
    ```
    
    <Accordion title="Platform-specific API URLs">
      - **iOS Simulator**: `http://localhost:8000/api`
      - **Android Emulator**: `http://10.0.2.2:8000/api` (Android maps localhost differently)
      - **Physical Device**: `http://YOUR_COMPUTER_IP:8000/api` (e.g., `http://192.168.1.100:8000/api`)
      - **Production**: `https://api.yourdomain.com`
    </Accordion>
  </Step>
  
  <Step title="Start the development server">
    Launch the Expo development server.
    
    ```bash
    npx expo start
    ```
    
    Or if you have Expo CLI installed globally:
    
    ```bash
    npm start
    ```
    
    This will start the Metro bundler and display a QR code in your terminal.
  </Step>
  
  <Step title="Run on your platform">
    Choose how you want to run the app:
    
    <Tabs>
      <Tab title="Web">
        Press `w` in the terminal or run:
        ```bash
        npx expo start --web
        ```
      </Tab>
      
      <Tab title="iOS">
        Press `i` in the terminal or run:
        ```bash
        npx expo start --ios
        ```
        
        <Note>Requires macOS with Xcode installed</Note>
      </Tab>
      
      <Tab title="Android">
        Press `a` in the terminal or run:
        ```bash
        npx expo start --android
        ```
        
        <Note>Requires Android Studio with an emulator or connected device</Note>
      </Tab>
      
      <Tab title="Physical Device">
        1. Install **Expo Go** from the App Store (iOS) or Google Play (Android)
        2. Scan the QR code displayed in your terminal
        3. The app will open in Expo Go
      </Tab>
    </Tabs>
  </Step>
</Steps>

## First Login

Once the app is running:

1. You'll see the **login screen** as the initial view
2. Enter your credentials (username and password)
3. The app will authenticate with your backend API at `/api/token/`
4. Upon successful login, you'll be redirected to the **profile page**
5. Use the bottom tab navigation to explore:
   - **Home**: Main dashboard
   - **Books**: View and manage books
   - **Authors**: View and manage authors
   - **Users**: View users (if you have admin access)
   - **Profile**: Your user profile and logout

<Tip>
  Make sure your backend API is running and accessible before attempting to log in.
</Tip>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Port already in use">
    If port 8081 is already in use, kill the process:
    
    ```bash
    # macOS/Linux
    lsof -ti:8081 | xargs kill -9
    
    # Windows
    netstat -ano | findstr :8081
    taskkill /PID <PID> /F
    ```
  </Accordion>
  
  <Accordion title="Unable to connect to API">
    - Verify your backend server is running
    - Check the API base URL in `api/axios.ts`
    - For Android emulator, use `http://10.0.2.2:8000/api` instead of `localhost`
    - For physical devices, use your computer's IP address
  </Accordion>
  
  <Accordion title="Metro bundler won't start">
    Clear the cache and restart:
    
    ```bash
    npx expo start --clear
    ```
  </Accordion>
  
  <Accordion title="Module not found errors">
    Delete node_modules and reinstall:
    
    ```bash
    rm -rf node_modules
    npm install
    ```
  </Accordion>
</AccordionGroup>
