<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl md:text-3xl font-semibold dark:text-white text-gray-900 mb-2">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get up and running with AppBiblioteca in minutes</p>
  
  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install all required packages.
      
      ```bash
      git clone https://github.com/ederjuarez/biblioteca_frontend-.git
      cd biblioteca_frontend-
      npm install
      ```
    </Step>
    
    <Step title="Configure your API endpoint">
      Update the API base URL in `api/axios.ts` to point to your backend server.
      
      ```typescript
      const api = axios.create({
        baseURL: "http://your-backend-url:8000/api",
      });
      ```
      
      <Note>The default configuration points to `http://localhost:8000/api`</Note>
    </Step>
    
    <Step title="Start the development server">
      Launch the Expo development server and run your app.
      
      ```bash
      npx expo start
      ```
      
      You can now open the app on:
      - **iOS**: Press `i` to open in iOS simulator
      - **Android**: Press `a` to open in Android emulator
      - **Web**: Press `w` to open in browser
    </Step>
    
    <Step title="Sign in and explore">
      Use your credentials to log in and start managing your library.
      
      The app will automatically handle token storage and refresh using `expo-secure-store`.
    </Step>
  </Steps>
</div>
